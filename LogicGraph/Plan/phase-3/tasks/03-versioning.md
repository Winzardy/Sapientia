# Таска 03 — retain/retire/remove (ОТМЕНЕНО)

**Статус: ✅ n/a (по ревью — из стореджа ничего не удаляем по одной)**

## Итог
refCount / Retain / Release / Remove(id) / retire / free-list — **отменены**. По решению ревью
(2026-06-08) `CompiledBlueprintStorage` ничего не освобождает по одной версии: версии только добавляются
и сосуществуют, все ресурсы освобождаются единым `Dispose` на выходе из приложения/PlayMode.

Удержание старой версии тривиально (она никогда не удаляется до Dispose). Управление тем, какую версию
использовать, — на стороне scope (Фаза 4), но storage версии не удаляет.
