# Таска 02 — dedup + сосуществование версий

**Статус: ✅ done (тесты 25/25 зелёные)**

## Итог
- **Dedup** по `(id, version)`: повторный `Add` той же пары не создаёт второй слот (входная арена
  освобождается, владелец — сторедж).
- **Сосуществование версий**: новая версия становится текущей (`RootSlot.slot`), прежняя уходит в
  `RootSlot.next` и продолжает жить. Lookup идёт по текущей, затем по списку старых.
- **Lazy-compile отменён** — компиляция эагерная/внешняя (`CompileLayout` у вызывающего), сторедж только
  хранит.
- **Ничего не удаляется по одной** — только `Dispose` на выходе.
